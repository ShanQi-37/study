# study

前端学习项目，包含：
- React + Vite 项目模板
- Tailwind CSS 登录页面实现
- frontend-design 技能包
- Git 提交规范配置

## 项目结构

```
├── src/                 # 源代码目录
│   ├── App.jsx         # 主应用组件
│   ├── main.jsx        # 应用入口
│   └── index.css       # 全局样式
├── skill/              # 技能包目录
│   ├── frontend-design.js  # 前端设计技能实现
│   ├── package.json    # 技能包配置
│   └── README.md       # 技能包文档
├── .trae/              # Trae IDE 配置
│   └── rules/          # 项目规则
│       └── git-commit-message.md  # Git 提交规范
├── index.html          # HTML 模板
├── package.json        # 项目依赖配置
├── vite.config.js      # Vite 配置
└── tailwind.config.js  # Tailwind CSS 配置
```

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- **React 18** - 前端框架
- **Vite 5** - 构建工具
- **Tailwind CSS 3** - CSS 框架
- **ESLint** - 代码规范检查

## 功能特性

✅ 现代化登录页面 UI
✅ 响应式设计，支持移动端
✅ 流畅的交互动画效果
✅ 完整的表单验证
✅ 可复用的 UI 组件库
✅ 标准化的 Git 提交规范
